RECENT WORK

GYM BUDDY

Gym buddy is a social networking platform for gym-goers. Through the app, individuals can create profiles containing their gym and availability information, and connect with other users, based on their profile details. The web application is built upon Ruby on Rails, interacts with the Yelp API, and is styled with ZURB Foundation.

EVEREST APP

Everest App is a long-term fitness goal tracker that integrates with the Fitbit device. Through the app, Fitbit users can create custom goals, and monitor their progress in real time. The application is built on Ruby on Rails, interfaces with the Fitbit and WolframAlpha APIs, and utilizes the morris.js library for charting the goal completion progress.

BIRDHAUS

Birdhaus is a data visualization dashboard, plotting multiple time series of temperature readings. The temperature data was collected over the time period of a week, via digital temperature sensors, connected to a RaspberryPi. The web application is built upon Ruby on Rails and AngularJS, and employs the Highcharts library for displaying the data.
